## Using go get:

`$ go get -u -v github.com/FallAngel1337/gobypass`


# How to use
| command | description|
| --- | --- |
| `gobypass -u <urls403.txt> -t 5 ` | Read the file using the flag with 5 threads|
| `cat urls403.txt \| gobypass -t 5` | Read the file from stdin with 5 threads|
